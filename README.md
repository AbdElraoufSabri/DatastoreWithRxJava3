This is an Android project allowing us to Use Datastore with RxJava3.

USAGE
-----

```groovy
implementation 'tech.abd3lraouf:datastore-rxjava3:1.0.0'
```
or

```kotlin
implementation("tech.abd3lraouf:datastore-rxjava3:1.0.0")
```
